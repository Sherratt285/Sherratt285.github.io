Control flow and loops, arrays and objects



flow control describes how a program preforms the job it is trying to do.

Its broken up into loops, functions, and condition statements. These are fundamentally the same across many different languages. from JavaScript to back-end languages such as the C family (C, C++, C# (C-sharp)) and Java as languages that are commonly used in the industry today.

Loops preform a set of instructions that is repeated while a condition is true.

There are two main types of loops found in programming. The For loop, and while loops.

The For loop runs a set of code for a set number of times. It can count up or down. It exits the loop when the index reaches the maximum or minimum value.

While loops are loops that loop while a condition is true. To implement a while loop, the test condition is initialised as the first step before the loop, and preformed as the last instruction in the loop. This stops the loop from looping indefinitely.

do while loops are slightly different. They are a while loop but they will always execute the code inside the loop on the first pass through before checking the loop condition.
